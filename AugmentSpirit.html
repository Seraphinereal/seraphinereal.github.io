Augment Spirit

Game Synopsis

Augment Spirit is a 2D fighting game made in C++ with SDL and OpenGL. Choose one of three characters, select an opponent, then fight till the victor is decided. At least that would've been the game if not for time constraints and being new to coding at the time. The game ended up only having one character finished, and a test dummy to use the moves on.

Characters


Raging Demon, a demonic brawler exerting black and red electricity. Frozen Sword, a refined swordsman with a cursed freezing blade. Abyssal Wretch, a kraken barely holding together a human form, its limbs shifting into massive tentacles for heavy hits.

These concepts will be revisited in future projects.

Experience

My goal was to pour my soul into this as my first solo project with visuals, however it instead became a series of lessons I would never forget. I learned far better time management, the importance of getting the core of the game done before taking any time on making things look presentable. The primary goal is a feature complete game. Always.

I fell so far behind during this project that I had to temporarily skip our final C++ project to work on Rekindle. I managed to catch back up shortly after though and breeze through a ton of work. I entered Augment Spirit naive, and I submitted the project humbled.
