Minesweeper

Game Synopsis

Minesweeper, made in C++ with SDL and OpenGL. The game consists of a grid of tiles, each of which being either a mine or not a mine. Safe tiles show a number counting all adjacent bombs to that tile. Using that information, flag unsafe tiles and reveal all safe tiles to win. Revealing a mine causes a game over, and the map can be reset by pressing the button at the top. There is a timer to check how fast it was completed, and a counter of how many flags there are left.

Experience

This was my first C++ game with visuals displayed instead of just text. I used free sprites off of a website for a game called KinitoPet, as I couldn't find the assets for the original windows minesweeper anywhere on the internet. At the time, making this project was a huge milestone for me, but looking back I have made so many bigger steps since then that I don't remember the struggles I faced.
